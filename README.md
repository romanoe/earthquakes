# earthquakes
An interactive visualization of Italian earthquakes from 1980 with d3.js. 


![earthquakes](https://user-images.githubusercontent.com/32194904/224549206-c02c736c-c9f8-43b1-ade4-a28d18099120.gif)


# Install

```shell

npm install

```

# Run

```shell

npm run dev

```


# Build

```shell

vite build

```
